# My Book Store

This project is an e-commerce website for selling books.

# Node JS, Express JS, and Mongoose

## Tools

- See [Node JS](https://nodejs.org/en/)

## Setup

1. `git clone https://github.com/rawda-developer/my-book-store.git`

### Run Backend

```
cd backend
npm install
npm run dev
```
